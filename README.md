# AutoHosts 🚀
自动解析 GitHub + Unraid 域名，生成加速 hosts
也可以解析其他域名，修改domain.list即可
---

# 加速地址
GitHub 原始地址
https://raw.githubusercontent.com/tee1975/autohosts/main/hosts

jsDelivr 加速地址
https://cdn.jsdelivr.net/gh/tee1975/autohosts@main/hosts

---

# 使用方法
SwitchHosts / 路由器 / NAS 都可以使用上面任意一个地址，自动更新加速。

---

# 如何添加新域名？
编辑仓库里的 domain.list文件，一行一个域名即可。
